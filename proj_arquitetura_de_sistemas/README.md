## Configurações KOTLIN

**Para instalar Kotlin no macOS, Linux, WSL:**

1. Instalar SDKMAN!

https://sdkman.io/

2. Para rodar via terminal um arquivo .kt

```shell
kotlinc MeuPrograma.kt -include-runtime -d MeuPrograma.jar
```

```shell
java -jar MeuPrograma.jar
```
